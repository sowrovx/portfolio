# Mahmudul Hasan Sowrov — Portfolio

A lightweight, responsive personal portfolio built as a single static HTML page.

## Edit the portfolio

Open `index.html` and search for `EDITABLE:`. Those comments mark the main sections you may want to change:

- Study details and availability
- Name and professional bio
- Projects, descriptions, years, and links
- LinkedIn, GitHub, and X links
- Contact email

The page styling is in the `<style>` section near the top of `index.html`.

## Preview locally

Open `index.html` in any browser. No build step or dependencies are required.

## Publish updates

Edit `index.html`, commit the change, and push it to GitHub. The connected hosting deployment will publish the new version automatically.
